# WBS Manager

ブラウザ単体で動作するプロジェクト管理ツールです。インストール不要のシングル HTML ファイルとして提供されます。OneDrive / SharePoint 同期フォルダを経由したチーム間のデータ共有にも対応しています。

**バージョン：v1.29.0**

---

## 特徴

- **インストール不要** — `wbs-tool.html` をブラウザで開くだけで即使用可能
- **ガントチャート** — 日 / 週 / 月 / 四半期（4月始まり年度）/ 年の 5 スケール。バーのドラッグで日程変更
- **階層構造** — フォルダとタスクを自由に組み合わせた無制限の階層管理
- **依存関係** — ドラッグ＆ドロップで前後工程を紐付け、重複時は赤線で警告
- **イナズマ線** — 今日時点の進捗遅れ・進みを視覚化。全スケールに追従
- **タスク非表示** — 完了済みタスクや確認不要なタスクを👁アイコンでワンクリック非表示化
- **ステータスフィルタ** — 「完了以外」を含む 6 種類のフィルタ
- **OneDrive 共有** — チェックイン / チェックアウト方式で複数人の同時編集を排他制御
- **自動ログイン** — 前回のストレージモード・ユーザーを記憶して起動時に自動ログイン
- **CSV 入出力** — 内部 ID 付きエクスポート、アップサートインポート対応

## クイックスタート

1. `wbs-tool.html` をダウンロードしてデスクトップに保存
2. Chrome または Edge でダブルクリックして開く
3. 起動ウィザードでストレージモードを選択
4. 前回ユーザーがある場合は自動ログイン → すぐに使い始めることができます

---

## 起動ウィザード

起動時に毎回ストレージ選択画面が表示されます。**通常はワンクリックで起動**できます。

```
起動
├─ [前回と同じ設定で続行] ──────────→ 自動ログイン（最速）
├─ [⚡ ワンクリックで再接続（OneDrive）] → OneDrive接続 → 自動ログイン
├─ [💾 ローカルストレージ] ──────────→ 自動ログイン
└─ [☁️ OneDrive（チーム共有）] ──────→ フォルダ選択 → 自動ログイン
```

前回ログインしたユーザーが有効な場合はユーザー選択画面を省略して自動ログインします。

## OneDrive 共有のセットアップ

### 1. SharePoint フォルダを OneDrive に同期する

1. SharePoint サイトの対象ライブラリを開く
2. WBS データ用フォルダを右クリック → **「OneDrive へのショートカットを追加」**
3. OneDrive デスクトップアプリで同期が完了したことを確認
4. チームメンバー全員が同じ手順を実施する

### 2. 初回接続

起動ウィザードで「☁️ OneDrive（チーム共有）」を選択 → フォルダ選択ダイアログで同期フォルダを指定

### 3. 2回目以降

起動ウィザードの「⚡ ワンクリックで再接続」ボタンをクリック → ブラウザの許可ダイアログに「許可」 → 自動接続・自動ログイン

## 編集モードと参照モード（OneDrive 使用時）

OneDrive モードでは起動直後は **参照モード（読み取り専用）** です。

| 操作 | 説明 |
|------|------|
| **✏️ 編集開始** | 最新データを読み込み、チェックイン（ロック取得）して編集可能にする |
| **✅ 編集完了** | データを保存し、チェックアウト（ロック解放）して参照モードに戻る |

**参照モード中は以下が無効化されます：**
タスクの追加・編集・削除・ドラッグ / 詳細パネルの入力 / プロジェクトの追加・削除 / ガントバードラッグ

30 秒ごとのポーリングで他ユーザーの編集状態とデータ更新を自動確認します。

## ガントチャートのスケール

| スケール | 内容 |
|---------|------|
| 日 | 1 日単位。土日は赤、Today は青でハイライト |
| 週 | 1 週単位（月曜始まり） |
| 月 | 1 ヶ月単位 |
| 四半期 | **4 月始まり年度**（Q1=4〜6月、Q2=7〜9月、Q3=10〜12月、Q4=翌年1〜3月） |
| 年 | 1 年単位 |

## CSV フォーマット

```
ID, No, タスク名, 種別, 成果物, ステータス, 担当者, 開始日, 終了日, 進捗率,
依存関係（ID）, 依存関係（No）, 親ID, 備考
```

- **ID 列あり** でインポート → アップサート（ID 一致タスクは更新、新規 ID は追加）または上書きを選択可能
- **ID 列なし**（旧形式）でインポート → 全体を上書き
