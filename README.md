# WBS Manager

ブラウザ単体で動作するプロジェクト管理ツールです。インストール不要のシングル HTML ファイルとして提供されます。OneDrive / SharePoint 同期フォルダを経由したチーム間のデータ共有にも対応しています。

**バージョン：v1.43.0**

---

## 特徴

- **インストール不要** — `wbs-tool.html` をブラウザで開くだけで即使用可能
- **ガントチャート** — 日/週/月/四半期（4月始まり）/年の5スケール。バーD&Dで日程変更、色分けルール、クリティカルパス、マイルストーン◆。ラベル列幅をドラッグで自由調整
- **ダッシュボード** — 健全性スコア・バーンダウン・遅延リスク・ワークロード分析など13種のカード。表示/幅/並び順をユーザーごとに設定可能
- **サイドバー統合** — プロジェクト切替・表示切替・フィルタ・今日のタスク・遅延タスクを1つのサイドバーで管理
- **フィルタ機能** — テキスト検索・担当者・自分のタスク・未完了・遅延・マイルストーンのみの6フィルタ。リストとガントに連動
- **変更履歴・Undo** — Ctrl+Z で最大30件Undo。変更ログを最大200件記録
- **OneDrive 共有** — チェックイン/チェックアウトで複数人の同時編集を排他制御
- **自動バックアップ** — 編集開始時に `Backup/` フォルダへ自動保存。最大10世代保持
- **改善要望管理** — ツール内でフィードバックと対応結果をチーム管理
- **自動ログイン** — 起動ウィザードが状況に応じた続行ボタン1つを表示。ワンクリックで前回の環境を復元

## クイックスタート

1. `wbs-tool.html` をダウンロードしてデスクトップに保存
2. Chrome または Edge で開く
3. 起動ウィザードでストレージモードを選択
4. 前回ユーザーがある場合は自動ログイン

---

## 起動ウィザード

前回の設定がある場合、状況に応じた続行ボタンが1つ表示されます。

| 続行ボタン | 条件 |
|-----------|------|
| ⚡ OneDriveに接続して続行（緑） | 前回OneDrive＋フォルダ記憶あり（最速） |
| ☁️ OneDriveモードで続行 | 前回OneDrive、フォルダ未記憶 |
| 💾 ローカルストレージで続行 | 前回ローカル |

初回起動時は「💾 ローカルストレージ」または「☁️ OneDrive（チーム共有）」を選択します。

## サイドバー

| セクション | 内容 |
|-----------|------|
| 📁 プロジェクト | セレクトで切替・✏️編集・🗑削除・＋新規 |
| 🖥 表示 | リスト／ガント／ダッシュボード／改善要望の切替 |

## OneDrive 共有

### セットアップ
1. SharePointの対象フォルダを右クリック → **「OneDriveへのショートカットを追加」**
2. 起動ウィザードで「☁️ OneDrive」→ フォルダ選択
3. チームメンバー全員が同じ手順を実施

### 2回目以降
「⚡ OneDriveに接続して続行」→ ブラウザの許可ダイアログに「許可」 → 自動接続

### ヘッダーの状態

| 表示 | 状態 | 編集 |
|------|------|------|
| 👁 参照モード（黄） | 起動直後 | ❌ |
| 🔒 ○○が編集中（赤） | 他ユーザー編集中 | ❌ |
| ✏️ 編集中（緑バッジ） | 自分が編集モード | ✅ |

### 自動バックアップ

「✏️ 編集開始」を押すたびに自動でバックアップを作成します。

```
WBSData/
  ├── wbs_data.json
  ├── wbs_feedback.json
  ├── wbs_lock.json
  └── Backup/
        ├── 2026-06-08_091500/
        │     ├── wbs_data.json
        │     └── wbs_feedback.json
        └── ... （最大10フォルダ）
```

復元方法：エクスプローラーで `Backup/` フォルダを開き、復元したいタイムスタンプフォルダの `wbs_data.json` をルートにコピーして上書きします。

## CSV フォーマット

```
ID, No, タスク名, 種別, 成果物, ステータス, 担当者, 開始日, 終了日, 進捗率,
依存関係（ID）, 依存関係（No）, 親ID, 備考
```

種別：`task` / `folder` / `milestone`、ステータス：`todo` / `doing` / `done` / `block`（英語表記・大文字小文字区別なし・日本語も対応）
