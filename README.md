# WBS Manager

ブラウザ単体で動作するプロジェクト管理ツールです。インストール不要のシングル HTML ファイルとして提供されます。OneDrive / SharePoint 同期フォルダを経由したチーム間のデータ共有にも対応しています。

**バージョン：v1.34.0**

---

## 特徴

- **インストール不要** — `wbs-tool.html` をブラウザで開くだけで即使用可能
- **ガントチャート** — 日/週/月/四半期（4月始まり年度）/年の5スケール。バーのD&Dで日程変更、色分けルール、クリティカルパス表示、マイルストーン◆
- **ダッシュボード** — 健全性スコア・バーンダウン・遅延リスク・ワークロード分析など13種のカード。カードの表示/非表示・幅・並び順をユーザーごとに設定可能
- **変更履歴・Undo** — Ctrl+Z で最大30件Undo。変更ログを最大200件記録
- **階層構造** — フォルダ・タスク・マイルストーンの無制限階層管理
- **タスク非表示** — 完了済みタスクをワンクリックで非表示化（データ保持）
- **OneDrive 共有** — チェックイン/チェックアウトで複数人の同時編集を排他制御
- **改善要望管理** — ツール内で改善要望と対応結果をチームで管理
- **自動ログイン** — 前回のストレージモード・ユーザーを記憶して起動時に自動ログイン
- **CSV 入出力** — 内部ID付きエクスポート、アップサートインポート対応

## クイックスタート

1. `wbs-tool.html` をダウンロードしてデスクトップに保存
2. Chrome または Edge で開く
3. 起動ウィザードでストレージモードを選択
4. 前回ユーザーがある場合は自動ログイン → すぐに使えます

---

## 起動ウィザード

```
起動
├─ [前回と同じ設定で続行] ──────── → 自動ログイン（最速）
├─ [⚡ ワンクリックで再接続（OneDrive）] → OneDrive接続 → 自動ログイン
├─ [💾 ローカルストレージ] ──────── → 自動ログイン
└─ [☁️ OneDrive（チーム共有）] ──── → フォルダ選択 → 自動ログイン
```

## ダッシュボード

「🏠 ダッシュボード」タブで全プロジェクト横断の状況を確認できます。

| カード | 内容 |
|--------|------|
| 🏥 健全性スコア | 0〜100点で進捗の健全性を評価 |
| 📉 バーンダウン | 理想vs実績の残タスク推移グラフ |
| ⚠ 遅延リスク | 計画より10%以上遅れているタスク一覧 |
| 📅 今週の作業予定 | 今日〜7日以内のタスク一覧 |
| ⚖ ワークロード | 担当者ごとの負荷状況 |
| 🔴 クリティカルパス | CPMで自動計算したCPタスク一覧 |
| 📊 フェーズタイムライン | フェーズ別の進捗横棒グラフ |

右上の「⚙ カード設定」ボタンでカードの表示/非表示・幅・並び順をユーザーごとに設定できます。

## OneDrive 共有

### セットアップ
1. SharePointの対象フォルダを右クリック → **「OneDriveへのショートカットを追加」**
2. 起動ウィザードで「☁️ OneDrive」→ フォルダ選択
3. チームメンバー全員が同じ手順を実施

### 2回目以降
起動ウィザードの「⚡ ワンクリックで再接続」→ 許可 → 自動接続・自動ログイン

### 参照モード・編集モード

| 操作 | 説明 |
|------|------|
| **✏️ 編集開始** | 最新データを読み込み、ロック取得して編集可能に |
| **✅ 編集完了** | データ保存・ロック解放 |

参照モード中は編集操作がUI・関数レベルの二重ガードでブロックされます。

## CSV フォーマット

```
ID, No, タスク名, 種別, 成果物, ステータス, 担当者, 開始日, 終了日, 進捗率,
依存関係（ID）, 依存関係（No）, 親ID, 備考
```

- **ID列あり** → アップサート（部分更新）または上書きを選択
- **ID列なし** → 全体を上書き
