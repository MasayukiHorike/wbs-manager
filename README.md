# WBS Manager

ブラウザ単体で動作する、プロジェクト管理のための WBS（Work Breakdown Structure）管理ツールです。インストール不要のシングル HTML ファイルとして提供されます。OneDrive / SharePoint 同期フォルダを経由したチーム間のデータ共有にも対応しています。

---

## 特徴

- **インストール不要** — `wbs-tool.html` をブラウザで開くだけで即使用可能
- **ガントチャート** — 日 / 週 / 月 / 四半期 / 年の 5 スケール切替、バーのドラッグで日程変更
- **階層構造** — フォルダとタスクを自由に組み合わせた無制限の階層管理
- **依存関係** — ドラッグ＆ドロップで前後工程を紐付け、重複時は赤線で警告
- **イナズマ線** — 今日時点の進捗遅れ・進みを視覚化
- **OneDrive 共有** — チェックイン / チェックアウト方式で複数人の同時編集を制御
- **CSV 入出力** — 内部 ID 付きエクスポート、アップサートインポート対応

## クイックスタート

1. `wbs-tool.html` をダウンロードしてデスクトップに保存
2. Chrome または Edge でダブルクリックして開く
3. サンプルデータが表示されます — すぐに使い始めることができます

## OneDrive 共有のセットアップ

チームでデータを共有する場合は以下の手順を実施してください。

### 1. SharePoint フォルダを OneDrive に同期する

1. SharePoint サイトの対象ライブラリを開く
2. WBS データ用フォルダを右クリック → **「OneDrive へのショートカットを追加」**
3. OneDrive デスクトップアプリで同期が完了したことを確認
4. チームメンバー全員が同じ手順を実施する

### 2. WBS Manager でフォルダを指定する

1. ヘッダーの「💾 ローカル」をクリック → ストレージ設定を開く
2. **「☁️ OneDrive（フォルダ指定）」** を選択
3. 「📂 OneDrive フォルダを選択」→ エクスプローラーで同期フォルダを選択
4. 「適用」をクリック

### 3. 2回目以降の起動

ログイン画面に **「⚡ ワンクリックで再接続」** ボタンが表示されます。クリックして許可するだけで接続完了です（フォルダ選択ダイアログは開きません）。

## 編集モードと参照モード（OneDrive 使用時）

OneDrive モードでは起動直後は **参照モード（読み取り専用）** です。

| 操作 | 説明 |
|------|------|
| **✏️ 編集開始** | 最新データを読み込み、チェックイン（ロック取得）して編集可能にする |
| **✅ 編集完了** | データを保存し、チェックアウト（ロック解放）して参照モードに戻る |

30 秒ごとのポーリングで他ユーザーの編集状態を自動確認します。

## CSV フォーマット

```
ID, No, タスク名, 種別, 成果物, ステータス, 担当者, 開始日, 終了日, 進捗率, 依存関係（ID）, 依存関係（No）, 親ID, 備考
```

- **ID 列あり** でインポートすると、ID 一致タスクは更新、新規 ID はタスク追加の **アップサート** が選択可能
- **ID 列なし**（旧形式）でインポートすると全体を上書き
