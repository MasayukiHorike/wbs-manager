# WBS Manager 設計書

## 概要

| 項目 | 内容 |
|------|------|
| ツール名 | WBS Manager |
| 形態 | シングル HTML ファイル（バックエンドなし） |
| 動作環境 | Chrome / Edge（File System Access API 対応ブラウザ） |
| データ保存 | localStorage / OneDrive 同期フォルダ（JSON） |
| 開発言語 | HTML / CSS / Vanilla JavaScript |

---

## 機能一覧

### 1. タスク・フォルダ管理

#### 1.1 階層構造

| 機能 | 概要 |
|------|------|
| 自由階層 | タスクとフォルダを任意の深さで入れ子にできる。階層数の制限なし |
| フォルダ | タスクをグループ化する入れ物。進捗率は子孫タスクから自動計算（期間加重平均） |
| タスク | 実作業単位。進捗率は手動入力 |
| 折りたたみ | フォルダ単位で子要素を折りたたみ・展開できる（リスト・ガント共通） |

#### 1.2 タスク属性

| 属性 | 概要 |
|------|------|
| 名前 | タスク・フォルダの名称 |
| 成果物定義 | そのタスクの完了条件・成果物を記述 |
| ステータス | 未着手 / 進行中 / 完了 / ブロック中 の 4 段階 |
| 担当者 | 登録ユーザーから選択、または直接入力 |
| 開始日 / 終了日 | YYYY-MM-DD 形式。子タスク追加時は親の日程を継承 |
| 進捗率 | 0〜100%（タスクのみ手動入力。フォルダは自動計算） |
| 依存関係 | 前工程タスクの ID を複数指定可能 |
| 備考 | メモ・補足情報 |

#### 1.3 CRUD 操作

| 操作 | 方法 |
|------|------|
| 追加 | ヘッダーボタン / 行左端の＋アイコン / 右クリックメニュー |
| 編集 | 行クリックで右パネル表示、フィールド直接編集 |
| 削除 | 行左端のゴミ箱アイコン / 右クリックメニュー（子孫も同時削除） |
| 並替 | 「⠿ 並替モード」ON 後、行左端ハンドルをドラッグ＆ドロップ |

---

### 2. リストビュー

| 機能 | 概要 |
|------|------|
| 階層インデント表示 | タスクの深さに応じてインデント表示。アイコンで種別（📁 / 📄）を識別 |
| 行番号（No） | 階層に応じた表示番号を動的計算（例：1, 1.1, 2.3.1） |
| 進捗バー | タスクは進捗率に応じたカラーバー。フォルダは自動計算値を表示 |
| 担当者アバター | イニシャルとユーザーカラーで視覚表示 |
| 期限ハイライト | 期限超過は赤文字、当日はオレンジで強調表示 |
| 依存関係表示 | 前工程の表示 No をチップ形式で表示 |
| フォルダ日程矛盾 | 子タスクの日程がフォルダ期間を超える場合に ⚠ マークと件数を表示 |
| 検索 | タスク名・成果物のテキスト検索 |
| フィルタ | ステータス・担当者でフィルタリング |
| 全展開 / 全折りたたみ | 一括操作ボタン |

---

### 3. ガントチャート

#### 3.1 表示スケール

| スケール | 列幅 | 上段ヘッダー | 下段ヘッダー |
|---------|------|------------|------------|
| 日 | 28px | 年月 | 日付（土日赤・Today青） |
| 週 | 56px | 年月 | 週開始日 |
| 月 | 112px | 年 | 月 |
| 四半期 | 224px | 年 | Q1〜Q4 |
| 年 | 336px | — | 年 |

タブ切替時は Today が画面横中央に自動スクロール。

#### 3.2 レイアウト固定

| 固定方式 | 対象 |
|---------|------|
| 左固定（sticky left） | タスク名列（横スクロール時も固定） |
| 上固定（sticky top） | ヘッダー行（縦スクロール時も固定） |
| 両方固定 | 交差点セル（タスク名ヘッダー） |

#### 3.3 バー操作（ドラッグ＆ドロップ）

| 操作 | 動作 |
|------|------|
| バー中央をドラッグ | 開始日・終了日を同時にシフト |
| バー左端をドラッグ | 開始日のみ変更 |
| バー右端をドラッグ | 終了日のみ変更 |

ドラッグ中はツールチップに日付と日数をリアルタイム表示。

#### 3.4 依存関係の矢印

| 種別 | 表示 |
|------|------|
| 正常 | グレー点線矢印 |
| 期間重複（前工程終了前に後工程開始） | 赤点線矢印 + ツールチップ警告 |
| 依存リンク中（ドラッグ中） | 青点線プレビュー矢印 |

#### 3.5 今日ライン

青い縦線で今日の日付を表示。上部に「TODAY」バッジ付き。今日列のセルも薄青でハイライト。

#### 3.6 イナズマ線

今日時点で期間が進行中または期限超過・未完了のタスクを対象として実績進捗点を縦につなぐ折れ線。

| マーカー | 意味 |
|---------|------|
| ◆ 青（始点・終点） | Today ライン上のアンカー点 |
| ● 青 | 今日ラインより右（進んでいる） |
| ● 赤 | 今日ラインより左（遅れている） |
| ◆ 赤 | 期間超過かつ未完了の遅延タスク |

完了済み・将来タスク・フォルダは除外。

#### 3.7 折りたたみ

ガントチャートのタスク名列でフォルダの ▼/▶ ボタンをクリックして個別折りたたみ。ヘッダーの ▼/▶ ボタンで全体の一括操作が可能。

### 4. 依存関係の管理

| 操作 | 方法 |
|------|------|
| 依存の追加（リスト） | 並替モード OFF 時に行をドラッグ → 別の行にドロップ |
| 依存の追加（ガント） | ラベル列の ⇢ をドラッグ → 別の行にドロップ |
| 依存の削除 | 既存の依存に再度ドロップ → 確認ダイアログで削除 |
| 直接入力 | 詳細パネルの「前タスク No」に表示番号をカンマ区切りで入力 |

---

### 5. プロジェクト管理

| 機能 | 概要 |
|------|------|
| 複数プロジェクト | プロジェクト単位でタスクを管理。プロジェクト間は独立 |
| プロジェクト作成 | 名前・説明・カラー・テンプレート（空白 or 基本 WBS）を指定して作成 |
| プロジェクト切替 | ヘッダーのドロップダウンまたはサイドバーのリストからワンクリック切替 |
| プロジェクト削除 | サイドバーのリストから削除（2 件以上の場合のみ） |
| 名前変更 | サイドバーのリストでインライン編集 |

---

### 6. ユーザー管理

| 機能 | 概要 |
|------|------|
| ユーザー登録 | 氏名・役割・メモ・アバターカラーを登録。上限なし |
| ログイン切替 | ログイン画面またはヘッダーのユーザーパネルからワンクリック切替（パスワードなし） |
| 自動ログイン | 初回起動時はデフォルトユーザーで自動ログイン |
| 担当者選択 | タスク追加・編集時に登録ユーザーから選択、または直接入力 |
| デフォルトユーザー | 田中 太郎・佐藤 花子・鈴木 一郎・山田 美咲（4 名）を初期登録 |

---

### 7. ストレージ管理

#### 7.1 ストレージモード

| モード | 保存先 | 特徴 |
|--------|--------|------|
| ローカルストレージ | ブラウザ内蔵 | 設定不要。単一 PC のみ |
| OneDrive | 指定フォルダの `wbs_data.json` | チーム共有可能 |

#### 7.2 OneDrive モードの動作

| ファイル | 用途 |
|---------|------|
| `wbs_data.json` | 全プロジェクト・ユーザーデータ |
| `wbs_lock.json` | 編集中ユーザー情報（チェックイン / アウト） |

- **参照モード（起動直後）**：読み取り専用。編集 UI が無効化される
- **チェックイン**：最新データを読み込み、ロックファイルに自分の情報を書き込む
- **チェックアウト**：データを保存し、ロックファイルをクリア
- **ポーリング**：30 秒ごとにロック状態とデータ更新を確認。参照モード中はデータ更新を自動リロード
- **フォルダハンドル永続化**：IndexedDB に `FileSystemDirectoryHandle` を保存し、次回起動時にワンクリックで再接続

### 8. CSV インポート / エクスポート

#### 8.1 エクスポート形式

```
ID, No, タスク名, 種別, 成果物, ステータス, 担当者,
開始日, 終了日, 進捗率, 依存関係（ID）, 依存関係（No）, 親ID, 備考
```

- ツリー順（表示順）で出力
- 内部 ID・表示 No・依存関係を内部 ID と表示 No の両方で出力

#### 8.2 インポート動作

| 条件 | 動作 |
|------|------|
| ID 列あり + 既存データあり | アップサート（更新）または上書きを選択 |
| ID 列なし（旧形式） | 全体を上書き |
| 日付フォーマット | YYYY-MM-DD 以外も Date オブジェクト経由で正規化 |

### 9. フォルダ日程矛盾の検知

子孫タスクの日程がフォルダの期間を超える場合を矛盾として検知し、以下の箇所に表示する。

| 表示箇所 | 内容 |
|---------|------|
| サイドバー | 矛盾件数カード（矛盾がある場合のみ表示） |
| リスト / フォルダ行 | ⚠ N 件バッジ + 薄赤背景。ホバーで詳細ツールチップ |
| リスト / タスク行 | ⚠ バッジ + 日付セルに赤背景 |
| ガント / ラベル列 | ⚠ アイコン + 薄赤背景 |
| ガント / バー | 赤いアウトライン枠線 |

矛盾の判定条件（全子孫を対象）：
- 子孫の開始日 ＜ フォルダの開始日
- 子孫の終了日 ＞ フォルダの終了日
- 子孫の開始日 ＞ フォルダの終了日

### 10. サイドバー

| 機能 | 概要 |
|------|------|
| プロジェクト一覧 | 全プロジェクトをリスト表示。完了数/総数付き |
| プロジェクト統計 | 総タスク数・完了タスク数・期限超過数・フォルダ日程矛盾件数 |
| ステータス凡例 | 各ステータスのカラー凡例 |
| 関係線凡例 | 通常（グレー）・期間重複（赤）の点線凡例 |
| 担当者一覧 | 登録担当者のアバターと名前 |
| 折りたたみ | ◀/▶ ボタンで最小幅に折りたたみ（ミニ統計表示付き） |

---

### 11. データ移行

| 機能 | 概要 |
|------|------|
| サンプルデータリセット | ストレージ設定モーダルからサンプルデータに戻す |
| 日付自動マイグレーション | 旧サンプルデータ（2025 年）を現在年に自動シフト（初回起動時のみ） |

## データモデル

### プロジェクト

```json
{
  "id": "p1",
  "name": "プロジェクト名",
  "desc": "説明",
  "color": "#2563EB",
  "tasks": [/* タスク配列 */]
}
```

### タスク

```json
{
  "id": "t12",
  "parentId": "t11",
  "type": "task | folder",
  "name": "タスク名",
  "deliverable": "成果物定義",
  "status": "todo | doing | done | block",
  "assignee": "担当者名",
  "start": "2026-05-12",
  "end": "2026-06-06",
  "progress": 55,
  "deps": ["t7", "t8"],
  "note": "備考"
}
```

### ユーザー

```json
{
  "id": "u1",
  "name": "田中 太郎",
  "role": "プロジェクトマネージャー",
  "note": "",
  "color": "#2563EB",
  "createdAt": "2026-01-01T00:00:00.000Z"
}
```

## localStorage / IndexedDB 管理

| キー | 種別 | 内容 |
|------|------|------|
| `wbs_manager_v1` | localStorage | 全プロジェクト・ユーザーデータ（キャッシュ） |
| `wbs_storage_mode_v1` | localStorage | ストレージモード・フォルダ名ヒント |
| `wbs_date_migrated_2026` | localStorage | 日付マイグレーション実行フラグ |
| `onedrive-folder` | IndexedDB | `FileSystemDirectoryHandle`（フォルダハンドル） |

## 技術構成

| 要素 | 内容 |
|------|------|
| UI フレームワーク | なし（Vanilla JS + CSS Variables） |
| フォント | DM Sans, DM Mono, Syne（Google Fonts） |
| ストレージ | localStorage, IndexedDB, File System Access API |
| グラフィック | SVG（ガント矢印、イナズマ線） |
| ビルドツール | なし（単一 HTML ファイル） |
| 外部依存 | Google Fonts のみ（オフライン環境では代替フォントにフォールバック） |
