📋 事前準備

推奨ブラウザ

Chrome または Edge での利用を推奨します。OneDrive連携（File System Access API）はSafariやFirefoxでは動作しません。

SharePointフォルダをOneDriveに同期する

1
SharePointサイトを開く

ブラウザで対象サイトにアクセスします。例：https://会社名.sharepoint.com/sites/プロジェクト名

2
WBSデータ用フォルダを用意する

ドキュメントライブラリでWBSデータを置くフォルダを選択、または「＋ 新規」→「フォルダー」で作成します（例：WBSData）。

3
「OneDriveへのショートカットを追加」をクリック

フォルダを右クリック → 「OneDriveへのショートカットを追加」を選択します。

4
OneDriveデスクトップアプリで同期を確認

タスクバーの雲マークをクリックし、同期完了を確認します。エクスプローラーにフォルダが表示されれば成功です。

5
チームメンバー全員が同じ手順を実施

全員が同じフォルダにショートカットを追加することで、全員のPCに同じフォルダが同期されます。

初回セットアップ

1
HTMLファイルをブラウザで開く

wbs-tool.html をデスクトップに保存し、Chrome または Edge でダブルクリックして開きます。

2
起動ウィザードでストレージを選択

「☁️ OneDrive（チーム共有）」を選択するとフォルダ選択ダイアログが開きます。手順4で同期したフォルダを指定してください。

3
ユーザーを選択してログイン

ユーザー選択画面が表示されます。未登録の場合は「新しいユーザーを追加してログイン」から登録してください。

💡 2回目以降は起動時に「⚡ OneDriveに接続して続行」ボタンが1つ表示されます。ブラウザの許可ダイアログに「許可」を押すだけで自動接続されます。

🚀 基本操作

タスクの種別

  • 📁 フォルダ：フェーズやグループを表します。進捗率は子タスクの期間加重平均で自動計算されます。
  • 📄 タスク：実際の作業単位です。進捗率を手動で入力します。
  • 🔷 マイルストーン：節目日を表します。開始日のみ設定。ガントチャートに◆ひし形マーカーで表示されます。

タスクの追加

操作方法
最上位タスクを追加ヘッダー左側の「＋ タスク追加」ボタン
同じ階層に追加行左端の同列アイコン（＋横線）
子タスクを追加行左端の子追加アイコン（青）
右クリックメニュー行を右クリック
💡 追加されたタスクはクリックした行のサブツリー直後に挿入されます。

タスクの非表示

行左端の👁アイコンで非表示化できます（データは保持）。サイドバーのフィルタ「🚫 非表示」チェックボックスで一覧確認・個別再表示が可能です。

並び替え

リストビュー上部の「⠿ 並替モード」をONにすると行をドラッグで並べ替えられます。ドロップ位置の上25%は同列上、下25%は同列下、中央50%は子として挿入されます。

⚠ 自分自身の子孫タスクへの移動は禁止されています。不正なドロップはトースト通知で警告されます。

サイドバーのフィルタ

フィルタ内容
タスク名検索タスク名・成果物に部分一致するタスクを表示
担当者特定の担当者のタスクのみ表示
自分のタスクログイン中のユーザーのタスクのみ表示
未完了のみ完了以外のタスクを表示
遅延のみ期限超過のタスクのみ表示
マイルストーンのみマイルストーン行のみ表示
💡 フィルタはリストビューとガントチャートの両方に連動します。プロジェクト切替時にはフィルタが自動リセットされます。

CSV インポート / エクスポート

  • エクスポート（ヘッダー右側の↑アイコン）：内部ID・No・依存関係・親IDを含むCSVを保存
  • インポート（ヘッダー右側の↓アイコン）：内部ID列があればアップサート（部分更新）または上書きを選択可能

種別・ステータスは英語表記で出力・入力します。大文字小文字は区別せず、日本語表記も認識します。

列値（英語）日本語でも認識
種別task / folder / milestoneタスク / フォルダ / マイルストーン
ステータスtodo / doing / done / block未着手 / 進行中 / 完了 / ブロック中
💡 タスク名や備考にカンマ・改行が含まれていても、CSVは正しくエスケープされるため列ズレは発生しません。

ヘッダー・ラベル列の固定

固定方向内容
縦スクロール日付ヘッダー（年月行・日付行）が上端に固定されたまま表示されます
横スクロールタスク名列（ラベル列）が左端に固定されたまま表示されます

ラベル列幅の調整

タスク名列の幅をドラッグで自由に変更できます。

  • ラベルヘッダーの右端にカーソルを合わせると ↔ カーソルに変化
  • そのままドラッグすると幅がリアルタイムに変化（160px〜600px）
  • マウスを離すと確定してガントを再描画

ツールバー（左から）

① 表示レンジ ② ⚡イナズマ線 ③ 🔴クリティカルパス ④ 色分けルール ⑤ 🖨印刷/PDF

バー色分けルール

ルール内容
ステータス未着手=グレー / 進行中=青 / 完了=緑 / ブロック=赤
担当者各ユーザーのアバターカラー
遅延状況完了=緑 / 期限超過=赤 / 遅れ気味=オレンジ / 順調=青

バー操作

  • バー中央をドラッグ：開始日・終了日を同時にシフト
  • バー左端をドラッグ：開始日のみ変更
  • バー右端をドラッグ：終了日のみ変更
💡 ドラッグ中はツールチップに日付と日数をリアルタイム表示します。参照モード中はバードラッグが無効化されます。

イナズマ線

今日時点の進捗遅れ・進みを実績進捗点を縦につないで可視化します。全スケールに追従します。

  • 🔵 青（今日ラインより右）：進んでいる
  • 🔴 赤（今日ラインより左）：遅れている

クリティカルパス

「🔴 クリティカルパス」ボタンで表示/非表示を切替。依存関係から最長経路（フロート=0のタスク）を自動計算します。ラベル列に赤い左ボーダー、バーに赤いアウトラインで強調表示されます。

マイルストーン

種別を「🔷 マイルストーン」にすると、ガントチャートに◆ひし形マーカーで表示されます。開始日のみ設定します（終了日は不要）。

☁️ OneDrive共有モード

起動ウィザード

起動時に前回の設定がある場合、状況に応じた続行ボタンが1つ表示されます。

表示されるボタン動作
⚡ OneDriveに接続して続行（緑）前回フォルダ記憶あり。ブラウザ許可ダイアログに「許可」を押すだけで自動接続（最速）
☁️ OneDriveモードで続行前回OneDriveを使用したがフォルダ未記憶。フォルダ選択ダイアログが開く
💾 ローカルストレージで続行前回ローカルを使用。そのまま起動
💾 ローカルストレージ（選択肢）ローカルモードで新たに起動
☁️ OneDrive（チーム共有）（選択肢）OneDriveモードで新たに起動。フォルダ選択ダイアログが開く

ヘッダーの状態表示

⚠ OneDriveモードでは起動直後は「参照モード」です。編集するには必ず「✏️ 編集開始」を押してください。
表示状態編集可否
👁 参照モード（黄バナー）参照モード（起動直後）❌ 不可
🔒 ○○が編集中（赤バナー）他ユーザーが編集中❌ 不可
✏️ 編集中（緑バッジ）自分が編集モード✅ 可能
  • ✏️ 編集開始：最新データを読み込み、自動バックアップを実行後、ロックを取得して編集可能になります
  • ✅ 編集完了：データを保存し、ロックを解放します。離席前に必ず押してください

📦 自動バックアップ

「✏️ 編集開始」を押すたびに、その時点のデータを自動的にバックアップします。

  • 保存先：OneDriveフォルダ内の Backup/ フォルダ（自動作成）
  • 形式：Backup/2026-06-08_143022_u1/ のようなタイムスタンプ＋ユーザーIDのフォルダ
  • 各フォルダに wbs_data.json（タスクデータ）と wbs_feedback.json（改善要望）を保存
  • 最大10世代保持。11件目が作成される際に最も古いフォルダを自動削除
  • バックアップ失敗時もトースト警告を表示し、編集モードは通常通り開始
💡 バックアップの wbs_data.json はツールで直接読み込める形式です。誤ってデータを壊した場合はエクスプローラーでファイルをルートにコピーして上書き復元できます。

参照モードで無効化される操作

タスクの追加・編集・削除・ドラッグ / 詳細パネルの入力 / プロジェクトの追加・編集・削除 / ガントバードラッグ / CSVインポート

自動更新

30秒ごとにOneDriveフォルダをポーリングし、他ユーザーの編集状態を自動確認します。参照モード中にデータが更新された場合は自動リロードされます。

サイドバーの折りたたみ

サイドバーの「◀」ボタンで折りたたみ、作業エリアを広げられます。折りたたみ中は📋📊🏠📝のアイコンボタンで表示を切り替えられます。

ダッシュボード（🏠 タブ）

13種のカードでプロジェクトの状況を可視化します。右上の「⚙ カード設定」ボタンで各カードの表示/非表示・幅（標準/2列）・並び順をユーザーごとに設定できます。設定はブラウザに保存されます。

変更履歴・Undo

ヘッダー左側の↩ボタン（または Ctrl+Z）で直前の操作を取り消せます（最大30件）。🕐ボタンで変更ログ（最大200件）を一覧表示できます。

バックアップからの復元（OneDriveモード）

データを誤って壊してしまった場合、エクスプローラーで Backup/ フォルダを開き、復元したい世代のフォルダ内の wbs_data.json を同期フォルダのルートにコピーして上書きします。改善要望も同様に wbs_feedback.json を上書きします。

フォルダ日程の矛盾検知

子タスクの日程がフォルダの期間を超えている場合、リスト・ガントチャート両方で ⚠ マークが表示されます。

改善要望管理（📝 タブ）

ツール内で改善要望と対応結果をチームで管理できます。OneDriveモードでは wbs_feedback.json として同フォルダに保存されます。

プロジェクト管理

サイドバーのプロジェクトセレクト横に3つのボタンがあります。

ボタン動作
✏️ 編集現在のプロジェクトの名前・説明・カラーを変更。参照モード中は無効
🗑 削除現在のプロジェクトを削除。2件以上のみ有効

「＋ 新規プロジェクト」ボタンで新規作成できます。プロジェクト切替時にフィルタは自動的にリセットされます。

タスク詳細パネルの閉じ方

詳細パネルが開いているとき、パネル左端の「›」帯をクリックするとパネルが閉じます。帯はパネル外にはみ出して常時表示されます。

CSVエクスポート形式

ID, No, タスク名, 種別, 成果物, ステータス, 担当者, 開始日, 終了日, 進捗率, 依存関係（ID）, 依存関係（No）, 親ID, 備考

内部IDが含まれるため、エクスポート→外部編集→インポートでアップサート（部分更新）が可能です。カンマや改行を含むフィールドも正しくエスケープされます。

データのリセット

ストレージ設定（ヘッダーのストレージ表示をクリック）の下部にある「🔄 サンプルデータでリセット」でデータを初期状態に戻せます。
